# Unnamed Roguelike

## Controls
To move:
- arrow keys
- numpad keys (diagonal support)
- vi keys (diagonal support)

To wait:
- .
- del
- keypad center

To exit:
- Q

To go back:
- backspace

To confirm:
- enter

To descend stairs:
- >

To ascend stairs:
- <

To pick up an item:
- p

To view/exit inventory:
- tab
- i

To go back to main menu:
- m

To drop item from inventory:
- d
